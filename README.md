# Factor Analysis of Turkey Life Index (R)

This project performs exploratory factor analysis on the 2015 Turkish Life Index dataset published by TURKSTAT.

The analysis includes:

- Data cleaning
- Missing value imputation
- KMO test
- Bartlett's Test
- Parallel Analysis
- Maximum Likelihood Factor Analysis
- Varimax Rotation
- Interpretation of factor loadings
- Exporting factor scores and loadings to Excel

Language: R
